This plugin shows a counter of pages/posts, all over words, comments, site impressions and unique visits in a widget.

== Description ==

This is a small wordpress plugin. It counts pages, posts, words, comments, site impressions and unique visits in your blog and displays it at a widget. Nothing less, nothing more.

You can change the displayed text in the yBlog Stats settings page.

== Installation ==
1. Upload or install it from Plugin Center.
2. Drag yBlog Stats Widget to a free space in your Widget Area.
3. Edit Options under Settings in your Admin Backend.
4. Have fun and share it to everybody.
